# Sales & Revenue Analysis Dashboard

## 📌 Project Overview
This project is an interactive Sales & Revenue Analysis Dashboard built using Microsoft Excel. It helps analyze sales performance using Pivot Tables, Pivot Charts, KPIs, and Slicers.

## 🎯 Objectives
- Analyze sales and revenue data
- Track key business KPIs
- Identify top-performing products
- Compare sales by region and category
- Visualize monthly revenue trends

## 📊 Dashboard Features
- Total Revenue KPI
- Total Orders KPI
- Total Quantity Sold KPI
- Average Revenue KPI
- Revenue by Product
- Revenue by Region
- Revenue by Category
- Monthly Revenue Trend
- Quantity Sold by Product
- Interactive Slicers (Product, Category, Region)

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## 🚀 How to Use

1. Open Dashboard.xlsx in Microsoft Excel.
2. Go to the Dashboard sheet.
3. Use the Product, Category, and Region slicers to filter the data.
4. Analyze the KPIs and charts.
